## Importante

Crear los roles con los que el sistema va a trabajar:

User roles:

| Rol   |      id      |
|----------|:-------------:|
| admin |  1 |
| user |    2   |
| delivery | 3 |

```sql
-- create roles
insert into tb_rol (id, name) values (1, 'admin'), (2, 'user'), (3, 'delivery');
```

```bash
npm install
npm run start:dev
```
